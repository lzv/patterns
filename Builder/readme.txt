Паттерн Строитель предназначен для поэтапного создания сложных объектов разных типов, с возможностью добавления новых. 
Например, нужно сконвертировать текст в разные форматы - html, plain, и тд. Создается интерфейс Builder, и для каждого формата создается наследник. Затем для имеющегося текста, который находится в исходном формате, создается парсер, которому передается объект строителя. 
В процессе разбора текста, парсер передает распознанные кусочки строителю, который собирает из них свой объект. После завершения у строителя запрашивается итоговый результат. 
Для добавления нового формата, в который будет преобразовываться текст, достаточно добавить нового наследника интерфейса Builder. Для нового формата исходного текста достаточно создать новый парсер, который будет работать с уже имеющимися строителями.
